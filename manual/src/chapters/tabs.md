# 5. 画面・タブ一覧

以下はメインシェルタブの代表一覧です。表示名は FXML の `text` に準拠しています。実際の並びは **タブ整理** の設定により異なります。

## 日常運用

| タブ名 | 主な用途 |
|--------|----------|
| 実行・ログ | 段階1/2 実行、ログ、出力パス、中断 |
| 配台計画_タスク入力 | 入力ブック・シート・表編集、段階2 起動前の入力 |
| 配台計画手動修正 | 配台結果 JSON の GUI 編集 |
| 設備ガント（グラフィック） | 設備別タイムライン表示・印刷向け |
| 納期管理ビュー | アラジン計画風の納期・実績比較 |
| オペレーターカード | 担当者カード表示 |
| 配台ワークスペース履歴 | スナップショット保存・復元 |

## バッジ設定（グループ）

| タブ名 | 主な用途 |
|--------|----------|
| UIバッジ | 画面上のバッジ部品のデザイン調整 |
| プッシュボタン | プッシュボタン見た目の調整 |
| 設備ガント・担当バッジ | ガント上の担当表示バッジ |

設計・調整用途が中心で、日常の配台実行では触れないことが多いです。

## 環境設定（グループ）

| タブ名 | 主な用途 |
|--------|----------|
| 環境変数 | `PM_AI_*` および計画用キーの編集 |
| メモリ設定 | JVM ヒープ、次回起動時の希望値 |
| グローバル設定 | UI 既定のリセット、パッケージ既定の書き出し |
| ユーザープロファイル | プロファイルの保存・読込 |

## タブ整理

メインタブの **順序**・**グループ見出し**・**色** を編集します。子タブ（TabPane 内）や TitledPane は参照行として表示され、ドラッグ対象外の場合があります。

<!-- MANUAL_SNAP:tabOrganizer -->

## 子タブ・折りたたみ

「配台計画_タスク入力」など、タブの内側にさらに **子タブ** や **TitledPane**（例: 操作・ソース / データ表）がある画面があります。タブ整理ツリーでは親タブの下にこれらの見出しが並びます。

運用ルール

1) 新規実装・修正は、巨大ファイルではなく上記 .bas で行う。
2) Excel へ反映するときは、VBE の「ファイル -> ファイルのインポート」で .bas を取り込む。
   （.bas は文字化け防止のため CP932 / Shift_JIS で保存する）
3) 反映後は VBE の「デバッグ -> VBAProject のコンパイル」を実行し、エラー 0 を確認する。
4) 巨大ファイル（生産管理_AI配台テスト_xlsm_VBA.txt）はバックアップ扱いにする。

# バンドルマニュアル（開発者向けメモ）

## 目的

JavaFX の「オンラインマニュアル」タブが読み込む HTML/CSS/画像です。オフラインで動作します。

## スクリーンショット運用（フェーズ2以降）

1. 同一テーマ（ライト推奨）・解像度を決め、キャプチャ規約をチームで固定する。
2. `images/` に `01-run-log.png` のように連番または章 ID（`ch02` 等）で保存する。
3. 各 HTML の `<img class="screenshot" …>` を実ファイル名に差し替える。
4. メインシェルのタブ見出し（`MainShell.fxml` の `text`）を変更したときは、同じ変更単位でマニュアル見出し・スクショを更新する。

## タブへジャンプ

本文からメインシェルのタブを開くリンクは次の形式です（許可されたキーのみ動作します）。

`pm-desktop://tab/<MainShellTabId.key()>`

例: `pm-desktop://tab/run` → 「実行・ログ」

キー一覧は `MainShellTabId.java` を参照してください。

オンラインマニュアル（開発者向け）

リポジトリ内のパス・エントリポイント索引です。巨大ファイルの全文検索は避け、付属のマップや検索ツールを使ってください。

はじめに・全体像

  • JavaFX エントリ: jp.co.pm.ai.desktop.PmAiFxApp
  • メインウィンドウ: MainShellController + fxml/MainShell.fxml
  • メインタブ ID: MainShellTabId.java（永続化キー）

実行・ログ

  • コントローラ: MainRunTabController.java
  • 段階スクリプト名定数: MainShellController 内 STAGE1 / STAGE2
  • タブ選択 API: MainShellController.selectMainShellTab(MainShellTabId)
  • 本マニュアルのタブリンクは pm-desktop://tab/<key> → 同 API にマップ（OnlineManualTabController）

環境設定（環境変数・メモリ・グローバル）

AppPaths.java が PM_AI_* を中心にパス解決します。EnvVarDocs で UI 説明と連動している場合があります。
